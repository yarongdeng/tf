**terraform 初始化**
```
terraform init
```


**terraform 计划输出**
```
terraform plan
```


**terraform 执行输出**
**修改配置，只需要修改完成main.tf之后再次执行terraform apply**
```
terraform apply
```

**terraform 销毁**
```
terraform destory
```

**有关tf docker provider 配置，可以参考[官方链接](https://www.terraform.io/docs/providers/docker/index.html)**
